# Nexus Tabletop

Framework modular para mesas virtuais (VTT) no Unity. Crie cenas, tokens e experiências de RPG de mesa digitais com foco em extensibilidade e rapidez.

## Roadmap
- [x] Estrutura `NexusUser` criada via menu do editor.
- [x] Cena principal (`MainScene`) e registro automático de subcenas.
- [x] Tokens billboard com sombra e states dinamicos.
- [x] Integração inicial com Mirror e componentes de rede.
- [x] Ciclo dia e noite & Clima via UI
- [x] Ui invisivel para softwares de captura
- [X] Camera Cinematica e Dinamica
- [ ] Salvamento/Carregamento do estado da mesa (persistência).
- [ ] Script de portas automatizado.
- [ ] Painel de controle de Cena
- [ ] Animated Tokens Suport
- [ ] Multiplayer 100% Funcional
- [ ] Cinematics em tempo real com cinemachine
- [ ] Camera que segue um token marcado
- [ ] Transição entre multiplas cameras e modos
- [ ] Prefab Token States


## Contribuindo
- **Issues**: reporte bugs, ideias e feedback.
- **Branches**: use `feat/`, `fix/`, `docs/`, `chore/` para organizar.
- **Commits** (sugestão): Conventional Commits (ex.: `feat(tokens): suporte a X`).
- **PRs**: inclua contexto, passos de teste e screenshots/gifs quando útil.

Sinta-se à vontade para abrir PRs pequenos e iterativos. Qualquer ajuda em documentação, exemplos e testes é muito bem-vinda.
